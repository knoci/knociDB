# knociDB
*A high-performance key-value storage database*

## Features
- ⚡ **High-performance** key-value storage
- 🔄 **Batch operations** with ACD guarantees
- 🌲 **B+Tree** and **Hash index** support
- 🌐 **Distributed deployment** with Raft protocol
